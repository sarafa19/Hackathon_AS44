Ce site a été conçu dans le cadre du Hackaton organisé par la 2ALSY ( Amicale des anciens du lycée scientifique de Yamoussoukro - Côte d'Ivoire )

Avec une équipe composée de DAO SEGUENON, KOUEHI JOEL, et moi même DIOMANDE Aminata Sarah, nous avons développé en 3 jours ce site.
L'implémentation du frontend et du Backend s'est faite avec Html, CSS, et PHP, puis la base de données SQL est administrée avec PHPMyAdminn
Il n'est pas encore terminé et nous y travaillons encore.

Le lien pour voir un aperçu: http://hackathon-as44.le-dolancourt.fr/acceuil.php
( Veuillez enlever le 's' du protocole de sécurité https si besoin )

Instruction pour avoir le programme:

- Importer la base de donnée annuaire;

- Configurer son ordinateur comme un serveur (Vous pouvez
utiliser xampp, wamp ...)

- Ouvrir le fichier Acceuil.php qui la page principale.
